Partie 1 : Téléchargement des fichiers nécessaires et installation des logiciels

Téléchargement de la distribution

Pour cette SAé nous avons utilisé la Distribution Linux nommé Ubuntu. Pour notre machine virtuelle nous utiliserons le logiciel VirtualBox sur Windows.

Télécharger l'image ISO d'Ubuntu 24.04 LTS à partir du site officiel : https://ubuntu.com/download/desktop

On se retrouve donc avec l'installateur de VirtualBox, ainsi que l'image disque d'installation de Ubuntu.

On procède a l'installation de VirtualBox.

Partie 2 : Création de la machine virtuelle

Une fois installé on arrive sur cette interface ou on va créer une nouvelle Machine Virtuelle en haut a gauche.

Sur cette première étape, on va nommé notre Machine, spécifié le système d'exploitation utilisé et donner notre image disque d'installation (précédemment téléchargé) a VirtualBox.

On choisi ensuite la quantité de RAM alloué a la machine virtuelle, 2048 Mo est un bon compromis pour une utilisation fluide.

Ici nous lui avons allouer : 4096 Mb de RAM ainsi que 3 Processeurs.

Ici, on fixera la taille maximum de notre Disque Dur virtuelle. On est partie sur 80Gio, au cas ou il nous est nécessaire de faire plus de choses dans la VM. On sait que cette espace n'est pas alloué directement sur le disque dur de notre hôte, mais uniquement lorsque notre machine virtuelle a besoin de plus de stockage.

Notre machine virtuelle est maintenant créer et prête a être démarré pour procéder au processus d'installation.

Partie 3 : Installation de la distribution

Ici, on installe Ubuntu

On démarre donc la machine virtuelle, et on aperçoit que GRUB démarre !

On arrive sur l'interface d'installation d'Ubuntu, on choisi la langue puis on clique sur Suivant.

On choisira l'installation par défaut, vu que ce n'est pas une machine qui a besoin de beaucoup de paquet.

On choisi d'effacer le disque et d'installer Ubuntu, vu que la machine virtuelle est neuve et n'a pas besoin de partitionnement spécifique.

Ici on va configurer notre utilisateur administrateur avec le modèle demander sur le document de référence.

L'installation est fini, on peu redémarrer la VM.

Partie 4 : Installation des additions

On peut se connecter à notre compte Administrateur.

Avant de faire quoi que ce soit, on va installer les Additions invités pour pouvoir utiliser le Drag n drop et le presse papiers.

On installe maintenant les dépendances nécessaires à l'installation des additions invités.

Ensuite on ouvrira le disque que l'on a monté dans la VM puis on ouvrira sa racine dans le terminal

On execute l'autorun dans le disque.

L'installation est lancé et s'est achevé.

Une fois que l'installation est fini on va éjecter le disque virtuel

Partie 6 : Installation et configuration du serveur web Apache2 puis déploiement

On va maintenant installer Apache2 sur la machine virtuelle. C'est Tiago qui s'en est chargé. Il s'ai donc connecté en tant qu'administrateur pour installer Apache2 avec la commande suivante :

sudo apt install apache2

À l’issue de l’installation, nous avons vérifié le bon fonctionnement du serveur Apache en accédant à l’adresse localhost : l’affichage de la page par défaut confirme que le service HTTP est correctement opérationnel.

Ensuite, nous avons Drag n drop notre site dans le dossier Documents de l'utilisateur administrateur

Nous avons ensuite copié les fichiers du site web dans le répertoire racine du serveur web Apache, qui est situé à l’adresse /var/www/html/

Enfin, nous avons vérifié le déploiement du site en accédant à l’adresse localhost dans un navigateur web.
