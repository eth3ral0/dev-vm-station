# SAE S1.03 - Installation d'un Poste pour le Développement

Site web documentant la **SAE S1.03** réalisée à l'**IUT Paris Cité - Rives de Seine** (Période B : Novembre 2025 - Janvier 2026).

## À propos

Compte rendu complet et interactif de la SAE "Installation d'un poste pour le développement" comprenant :
- Explication de la virtualisation et de ses concepts
- Guide de création d'une machine virtuelle (VirtualBox + Linux, en l'occurence ici Ubuntu)
- Configuration système (utilisateurs, groupes, permissions)
- Déploiement d'un serveur web Apache
- Planning et organisation du projet (Diagramme de Gantt)
- Rapport documenté avec justification des choix techniques


## Objectifs

- S'initier à la virtualisation avec VirtualBox
- Installer et configurer un système d'exploitation Linux
- Gérer les utilisateurs, groupes et permissions
- Mettre en place un serveur web Apache
- Déployer une documentation web responsif (HTML5/CSS3/Bootstrap)
- Documenter et justifier tous les choix techniques

## Accédez au Site

Le site est hébergé sur ce repository et présente :
- **Contexte & Objectifs** - Comprendre la SAE
- **Organisation du Projet** - Planning et répartition
- **Installation de la VM** - Guide complet étape par étape
- **Rapport de Projet** - Résultats et justifications
- **Questions** - Réponses détaillées aux questions de la SAE
- **Ressources** - Liens et références utilisés

## Technologies

- **Virtualisation** : Oracle VirtualBox
- **Système** : Linux, Ubuntu
- **Serveur Web** : Apache HTTP Server
- **Frontend** : HTML5, CSS3, Bootstrap 5
- **Outils** : Visual Studio Code, Git/GitHub, PhpStorm
